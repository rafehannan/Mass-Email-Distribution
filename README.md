# Mass-Email-Distribution
Script for sending HTML email to large list of volunteers for onboarding.
SETUP INSTRUCTIONS:
1. Open your Google Sheet with volunteer data
2. Go to Extensions > Apps Script
3. Delete any existing code and paste this entire script
4. Save the project (name it "Email Campaign")
5. Run the function "sendWaterWiseEmails" from the toolbar
6. Grant necessary permissions when prompted
 
SPREADSHEET REQUIREMENTS:
- Column A: "Full Name" (e.g., "John Smith")
- Column B: "Official Email Address"
- Data should start from Row 2 (Row 1 is headers)


Run script ONCE to authorize Drive access
This forces Google to ask for permissions
